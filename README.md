# Menu Tune

![Platform](https://img.shields.io/badge/platform-macOS-lightgrey) ![License](https://img.shields.io/badge/license-MIT-blue)

**Show Now Playing music in your menu bar.**

Menu Tune is a minimal macOS utility that displays the currently playing track and artist directly in your menu bar.

## Features

- **Now Playing**: Shows artist and title information.
- **Playback Control**: Control Spotify and Apple Music.
- **Customizable**: Compact mode, adjustable styles, and themes.
- **Launch at Login**: Start automatically when you log in.

## Requirements

- macOS 15.0+

## Development

1. Open `MenuTune.xcodeproj` in Xcode.
2. Press `Cmd+R` to Build and Run.
